# Introduction

Affiche et modifie les tables de traduction d'adresses IP en adresses
physiques utilisées par le protocole de résolution d'adresses ARP.

# Synopsis

`arp [paramètres]`

# Options

| Raccourci                       | Description                         |
| ------------------------------- | ----------------------------------- |
| `-a`                   | Affiche les entrées ARP en cours         |
