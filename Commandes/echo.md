# Introduction

Affiche des messages ou active/désactive l’affichage des commandes.

# Syntaxe

`echo [options]`

# Options

| Option      | Description                                   |
| ----------- | --------------------------------------------- |
| `[ON/OFF]`  | Active ou désactive l'affichage des commandes |
| `[message]` | Affiche un message                            |

# Exemples

Affiche le nom de l'utilisateur actuel

`echo %USERNAME%`
